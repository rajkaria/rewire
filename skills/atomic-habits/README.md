# Atomic Habits Coaching

**Source:** Atomic Habits by James Clear

**Domain:** Habit formation, behavior design, identity change

**Depth:** Flagship

## What This Skill Does

Coaches you through building lasting habits using a structured, identity-first approach to behavior change. Instead of relying on motivation and willpower, the agent helps you design systems — the cues, environments, routines, and feedback loops — that make good habits automatic and bad habits difficult. You'll move from understanding why habits work to running real experiments in your life to maintaining self-sustaining systems you designed yourself.

## How to Use

1. Load `SKILL.md` into your AI agent (Claude Code, Claude.ai, OpenClaw, Cursor, or any LLM)
2. Start a conversation — the agent will guide you from your first session
3. Commit to the experiments between sessions — real-world action is where the change happens
4. Check in regularly — the agent tracks your commitments and follows up

## Lessons

| # | Lesson | Description |
|---|--------|-------------|
| 1 | Identity | Why identity-based change works and how to define who you want to become |
| 2 | Habit Loop | The cue-craving-response-reward loop and how to see it in your own behavior |
| 3 | Make It Obvious | Designing cues and using implementation intentions to trigger habits |
| 4 | Make It Attractive | Temptation bundling, motivation rituals, and the role of craving |
| 5 | Make It Easy | Reducing friction, the two-minute rule, and environment design |
| 6 | Make It Satisfying | Immediate rewards, tracking, and the dopamine feedback loop |
| 7 | Breaking Bad Habits | Inverting the four laws to make unwanted habits invisible, unattractive, difficult, and unsatisfying |
| 8 | Mastery and Systems | Moving from single habits to interconnected systems, advanced strategies, and long-term maintenance |

## Exercises

| Exercise | Type | Description |
|----------|------|-------------|
| Habit Audit | reflection | Map your current daily habits and identify cues, routines, and rewards |
| Identity Declaration | reflection | Define the identity you want to build and connect it to specific habits |
| Implementation Intentions | planning | Design specific when/where/how plans for your target habits |
| Habit Stacking | planning | Link new habits to existing routines using the habit stack formula |
| Environment Design | experiment | Reshape your physical or digital environment to support your habits |
| Two-Minute Scaling | experiment | Scale your target habit down to a two-minute entry version and test it |
| Habit Scorecard | assessment | Score your daily behaviors as positive, negative, or neutral and identify patterns |
