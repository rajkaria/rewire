# Check-In Protocol

## Frequency
Weekly. If no session in 10 days, reach out proactively.

## Opening
Agent says: "Good to see you back. Let's check in on how things are landing. I want to hear about both the wins and the walls — both are useful data."

## Core Questions
1. "What values-aligned actions did you take this week? Name them specifically — even small ones count."
2. "Did you notice any hook thoughts trying to pull you off course? What did they sound like, and how did you respond?"
3. "Was there a moment where you chose avoidance over action? What got in the way?"
4. "On a scale of 1-10, how much are you living in line with what matters to you right now — compared to when we started?"

## Scoring (Internal)
The agent assesses progress across four dimensions — not shared with user directly:
- **Engagement**: Are they doing the exercises and experiments between sessions?
- **Defusion skill**: Can they notice thoughts without being controlled by them?
- **Values clarity**: Can they articulate what matters in their own words?
- **Behavioral flexibility**: Are they making different choices — choosing values over avoidance?

## Adaptation
- If progress is strong: Advance to the next lesson or assign a more challenging committed action. Introduce compound actions that serve multiple values simultaneously.
- If progress is stalled: Check whether the committed actions are too big. Shrink them. Revisit defusion — the user may be fused with thoughts about the process itself ("this isn't working," "I'm not doing it right").
- If user is struggling: Slow down. Focus on one value and one tiny action. Rebuild momentum with micro-wins before expanding scope. Check for red flags if distress is escalating.
