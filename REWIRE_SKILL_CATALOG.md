# Rewire — Comprehensive Skill Catalog

> A curated list of 100+ potential coaching skills across ancient wisdom, world religions, philosophy, modern psychology, applied life domains, and emerging frameworks. Organized by category with source material, coaching domain, and composability notes.

---

## How to Read This Document

Each skill entry includes:
- **`skill-name`** — The proposed folder name
- **Source** — The primary text, tradition, or author
- **Domain** — What it coaches (e.g., emotional resilience, decision-making)
- **Why it fits Rewire** — What makes it coachable via AI (not just informational)
- **Composability** — How it connects to other skills

Skills marked with ⭐ are high-priority recommendations beyond the existing 12.
